## 📌 Overview

This repository contains a collection of **beginner-friendly Python projects** built to strengthen core programming skills through hands-on practice.

The projects focus on:

* Problem solving
* Logic building
* Data structures (lists, dictionaries)
* Loops and conditionals
* User input handling
* Basic game development

---

## 🚀 Projects Included

### 🚗 Car Game (Console App)

A simple command-based driving simulator where the user can start, stop, or exit the car system.

**Key Concepts:**

* While loops
* Conditional statements
* User commands

---

### 😀 Emoji Translator

Converts text-based emoticons into emojis.

**Example:**

```
Input:  Hello :) I am sad :(
Output: Hello 😀 I am sad 😞
```

**Key Concepts:**

* Dictionaries
* String manipulation
* Input parsing

---

### 🔺 Shape Printer

Generates custom patterns based on user input.

**Example:**

```
Input: 3 → [1, 3, 2]

Output:
x
xxx
xx
```

**Key Concepts:**

* Lists
* Loops
* Pattern generation

---

### 🎯 Number Guessing Game

A game where the computer selects a random number and the user tries to guess it.

**Features:**

* Random number generation
* Hints (higher/lower)
* Attempt counter

**Key Concepts:**

* While loops
* Random module
* Condition tracking

---

### 🔢 Number to Words Converter

Converts digits into words.

**Example:**

```
123 → one two three
```

**Key Concepts:**

* Dictionaries
* String iteration

---

### 📋 Duplicate Remover

Removes repeated values from a list.

**Key Concepts:**

* Lists
* Loop filtering
* Data cleaning logic
